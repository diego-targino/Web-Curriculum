Deck Builder (Em Breve)

Descrição:

Este projeto pessoal consistem em uma Web API Restful integrada a um sistema react cujo objetivo é criar um ambiente para a criação e edição de Decks de Yu-Gi-Oh.

Tecnologias:

Toda a parte visual do projeto está sendo feita utilizando React TS seguindo o template do Vite.

A Web API Restfull está sendo desenvolvida ultilizando C# com a versão 8 do .Net Framework e o Pacote Entity Framework para manipulação do banco de Dados.

O banco de dados utilizado atualmente no projeto é o PostgreSQL.

Prazo para a finalização:

Todo o desenvolvimento da Web API está presvisto para ser concluído por meados de 07/2024.
